# 📌 About SmartHire

SmartHire is a modern AI-powered recruitment platform designed to simplify and automate the hiring process for both recruiters and job seekers.

The platform uses ATS-based resume analysis, intelligent candidate ranking, resume parsing, and AI-driven job matching to help recruiters identify the best candidates quickly and efficiently.

---

# 💡 Why SmartHire?

Traditional hiring systems are slow, repetitive, and inefficient.

Recruiters often receive hundreds of resumes for a single job opening, making manual screening difficult and time-consuming.

SmartHire solves this problem using:
- AI-based Resume Analysis
- ATS Score Calculation
- Candidate Ranking System
- Smart Resume Parsing
- Automated Shortlisting Emails

This helps recruiters hire smarter and faster.

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/SmartHire.git

cd SmartHire
```

---

## 2️⃣ Frontend Setup

```bash
cd frontend

npm install

npm run dev
```

---

## 3️⃣ Backend Setup

```bash
cd backend

npm install

npm start
```

---

# 🔐 Environment Variables

Create a `.env` file inside the backend folder.

```env
PORT=5000

MONGO_URI=your_mongodb_connection

JWT_SECRET=your_secret_key

OPENAI_API_KEY=your_openai_api_key

EMAIL_USER=your_email

EMAIL_PASS=your_email_password
```
